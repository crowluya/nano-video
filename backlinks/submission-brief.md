### Submission Workflow
1. Use Playwright browser tools for automation
2. Source list: `backlinks/150个外链.txt`
3. Skip sites already processed in `submission-tracker.md`
4. Update tracker after each submission attempt

### Handling Rules
| Scenario | Action |
| --- | --- |
| Already processed | Skip |
| Requires payment | Mark as `Follow Up` with "Paid only" note |
| Requires login | Use Google login, then continue |
| Forum/Community | Mark as `Follow Up`, skip to next |
| Cloudflare block | Mark as `Follow Up` with "Cloudflare block" note |
| Success | Mark as `Submitted` |
