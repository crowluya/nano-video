# SEO Internal Link Language Rollout

Created: 2026-03-15

## First Batch

The first internal-link rollout is now localized for these 4 locales:

- `en` - English
- `zh` - Chinese
- `ja` - Japanese
- `es` - Spanish

## Covered Scope

This first batch includes localized content for:

- Header `Explore` and `Guides` navigation entries
- Mobile menu `Explore` and `Guides` entries
- Footer injected internal-link entries
- Homepage internal-link modules
- Tool-page internal-link modules
- Blog list and blog detail internal-link modules
- 6 P0 SEO landing pages
- 3 guide pages

## Fallback Rule

Other locales currently fall back to English for the new internal-link system.

This means:

- Existing site translations outside the internal-link rollout still follow the current locale files
- New SEO/internal-link content is fully localized only for `en`, `zh`, `ja`, and `es`
- Remaining locales should be added in the next language expansion batch
