使用 Playwright MCP 执行以下任务：

1. 打开网页: $ARGUMENTS
2. 等待页面加载完成
3. 获取页面快照确认加载成功
4. 截取完整页面截图
5. 关闭浏览器

请按步骤执行并报告结果。
