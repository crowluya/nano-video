# 测试指令

请执行以下检查步骤，确保代码质量。

## 执行步骤

1. **Lint 检查**
   ```bash
   !pnpm lint
   ```

2. **类型检查**
   ```bash
   !pnpm build
   ```

3. **数据库检查**（如涉及 schema 变更）
   ```bash
   !pnpm db:push
   ```

4. **i18n 一致性检查**
   - 检查新增的翻译 key 在 en/zh/ja 三个语言中都存在
   - 检查是否有硬编码文本
